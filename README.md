# Digital Library

Simple static web project: index page with styles and script.

## Structure

- `index.html` 
- `css/style.css`
- `js/script.js`

## Usage

1. Open `index.html` in a browser.
2. Use the app features (add/search books, etc.) as implemented.

## Development

- Edit HTML in `index.html`.
- Edit CSS in `css/style.css`.
- Edit JavaScript in `js/script.js`.

## Notes

- No build tools required.
- Works offline.
